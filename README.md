# Tentame Kelzer

Tentame Kelzer é um projeto desenvolvido por Gabriel Kelzer utilizando Shell Script no Termux. O objetivo da ferramenta é demonstrar técnicas de síntese e manipulação de áudio para fins educacionais, automação e estudos relacionados à tecnologia mobile e segurança digital.

O projeto utiliza o programa `espeak` para transformar texto em voz diretamente pelo Termux, criando áudios personalizados de forma rápida e prática.

---

## ⚙️ Funcionalidades

- Gerar voz masculina
- Gerar voz feminina
- Integração com `espeak`
- Interface personalizada em ASCII
- Compatível com Termux
- Script leve e simples de utilizar

---

## 📱 Requisitos

- Termux
- Git
- Bash
- Espeak
- Conexão com internet

---

## 🚀 Instalação

```bash
termux-setup-storage
apt update 
apt upgrade 
pkg install python
pkg install pip
pip install --upgrade pip
pip install lolcat
pkg install espeak
cd
ls
git clone https://github.com/gabrielkelzer/Tentame_Kelzer
cd Tentame_Kelzer
chmod +x *
. ./Tentame_Kelzer.sh
```

---

## 📖 Como usar

Após abrir o script, escolha uma das opções disponíveis no menu principal.

```text
[1] Como funciona a script
[2] Enviar voz de homem
[3] Enviar voz de mulher
[4] Sair
```

Basta selecionar a opção desejada e seguir o passo a passo exibido na tela.
